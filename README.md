# 🌍 imbonicard.com Website

**imbonicard.com** is a modern digital platform designed to help organizations **monitor development programs, manage beneficiaries, and measure impact** across multiple sectors including agriculture, environmental initiatives, green energy, and community development.

This repository contains the **official marketing website and frontend interface** for imbonicard.com.

---

## 🚀 Overview

imbonicard.com provides organizations with the tools they need to:

* Register and manage beneficiaries
* Track field activities and project participation
* Monitor program implementation in real-time
* Collect structured data from the field
* Generate insights and measure impact

The website serves as:

* A **product showcase**
* A **partner acquisition platform**
* A **gateway to the imbonicard.com application**

---

## 🧱 Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** Tailwind CSS
* **Deployment:** Vercel (recommended)
* **Forms:** Formspree (temporary contact handling)

---

## 📬 Contact Form (Temporary Setup)

The contact form is currently powered by **Formspree** to allow message delivery without a backend.

### How it works:

* User submits form
* Data is sent to Formspree endpoint
* Messages are delivered directly to the configured email inbox

### Future Plan:

This will be replaced with a custom backend endpoint:

```
POST /api/contact
```

## 🛠️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/tuyizereerneste/agri-new.git
cd imbonicard.com-website
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run development server

```bash
npm run dev
```

---

### 4. Build for production

```bash
npm run build
```

---

## 🌍 Deployment

### Recommended: Vercel

* Connect your GitHub repository
* Automatic deployments on push
* Built-in HTTPS and CDN

---

### Domain Setup

Recommended domain structure:

* `imbonicard.com` → Website
* `app.imbonicard.com` → Application frontend
* `api.imbonicard.com` → Backend API
